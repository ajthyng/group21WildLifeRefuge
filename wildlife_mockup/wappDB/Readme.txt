
How to create a local db and windows forms app

 to set this up for microsoft's SQL
 Solution Explorer:  right-click on project > Add > Item > C# > data > Service-based Database
 project>properties   > security >  set "This is a full trust application"

OR

To add DB to VS2017, right-click the "Data-Connections" in Server Explorer. 
Select "Add Connection". Provide name for new DB and the .....mdf file will be created.
